# XestiTokens

A rules-based lexical tokenizer.

## <a name="overview">Overview</a>

The XestiTokens framework provides a highly configurable rules-based lexical
tokenizer written in Swift. You can use it to tokenize a wide variety of text
input.

> “If you can tokenize it with `lex` or `flex`, you can tokenize it with
> XestiTokens.”
